# 📡 iptvchannel Channel Dataset
This repository contains the official public dataset used by **Famelack**, including IPTV streams and YouTube/Twitch live channels from around the world.

Famelack makes discovering working live channels simple, fast, and enjoyable. No accounts, no installs, no hassle.

## 🌍 What this dataset includes
- Channels from more than 100 countries  
- IPTV and YouTube live sources  
- Language, country, and category metadata  
- Organized JSON files by:
  - `compressed/`
  - `raw/`

## 🛠 How the data is maintained
This dataset is automatically generated by the Famelack ingestion and validation pipeline. The process includes:

- Daily checks of channel uptime and reliability  
- Removal of broken or non-working streams  
- Monitoring active YouTube live channels  
- Adding newly discovered and verified streams  
- Respecting DMCA and rights-holder requests  

**If a stream doesn’t work reliably, it doesn’t stay.**

## 🙏 Upstream acknowledgment
CkIPTV benefits from the work of the open-source community.  
A significant portion of our discovery workflow is sourced from the excellent **iptv-org** community list, which we continually refine through:

- Automated validation and uptime monitoring  
- CORS and playback compatibility checks  
- Removal of broken or duplicate streams  
- Corrected metadata and standardized country assignment  
- Integration of YouTube live channels  
- Addition of sources not found upstream  

This repository is **not a mirror** — it is continuously transformed, filtered, and enriched to support the iptvchannel platform.

## 📌 About Famelack as a source
This repository powers **Famelack.com**, which consumes these channel files as its primary data source.

The website provides the full interactive experience built on top of this dataset — including search, playback integration, favorites, globe navigation, and platform features.

## 📁 Folder structure
```text
channels/
├── compressed/
│   ├── countries/
│   ├── categories/
│   └── countries_metadata.json
└── raw/
    ├── countries/
    ├── categories/
    └── countries_metadata.json
```

Metadata files such as countries_metadata.json exist in both compressed and raw.

JSON field names and structure may evolve without notice as we optimize for the Famelack platform.

**Hotlinking**

For production use, we encourage caching or maintaining your own mirror to avoid possible service interruptions.
